# Docker Develop Environment

<h2 align="center"> On Laptop </h2>

- Build yourself

```
$ cd ~/project_seadrone/docker/laptop
$ source docker_build.sh
```

- Run docker
```
$ cd ~/project_seadrone
$ source docker/laptop/docker_run.sh
```

- Join docker
```
$ cd ~/project_seadrone
$ source docker/laptop/docker_join.sh
```
